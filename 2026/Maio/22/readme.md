# 🗂️ Verificador de Arquivos Antigos

Este script lista todos os arquivos de uma pasta que não foram modificados há mais de 6 meses, ajudando a identificar arquivos desatualizados ou potencialmente inativos.

## ✨ Funcionalidades
- **Identificação de Arquivos Velhos:** Encontra arquivos que não foram modificados há mais de 6 meses.
- **Listagem de Arquivos:** Exibe a lista de arquivos desatualizados no terminal.
- **Uso Simples:** Executa rapidamente e sem necessidade de configurações adicionais.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Ele utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_arquivos_antigos.py
```
