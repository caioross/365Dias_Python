# 🧮 Calculadora de Equação Segundo Grau

Este script resolve equações de segundo grau utilizando a fórmula de Bhaskara, solicitando ao usuário os coeficientes A, B e C.

## ✨ Funcionalidades
- Calcula as raízes de uma equação do segundo grau.
- Solicita os coeficientes A, B e C ao usuário.
- Trata casos de equações com raízes reais e complexas.
- Exibe os resultados de forma clara e formatada.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_equacao_segundo_grau.py
```
