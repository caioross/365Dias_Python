# 🎬 Extrator de Tags YouTube

Este script recebe a URL de um vídeo do YouTube e extrai as palavras-chave (tags) usadas pelo criador, facilitando a análise e organização de conteúdo.

## ✨ Funcionalidades
- Recebe uma URL de vídeo do YouTube como entrada.
- Extrai as tags (palavras-chave) associadas ao vídeo.
- Exibe as tags no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `requests` para fazer requisições HTTP.
- Biblioteca `beautifulsoup4` para parsing do HTML.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python extrator_tags_youtube.py
```
