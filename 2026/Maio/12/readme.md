# 🔄 Conversor de Moedas em Tempo Real

Este script permite a conversão de valores entre Real (BRL), Dólar (USD) e Euro (EUR) utilizando cotações atualizadas da internet, facilitando a gestão financeira e a comparação de valores em diferentes moedas.

## ✨ Funcionalidades
- **Conversão de Moedas:** Converte valores de Real para Dólar, Euro, e vice-versa.
- **Cotações Atualizadas:** Utiliza dados em tempo real da internet para garantir precisão nas conversões.
- **Interface Simples:** Interface de linha de comando para facilitar a utilização.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **biblioteca `requests`**: Utilizada para buscar cotações atualizadas da internet.
- **biblioteca `json`**: Utilizada para processar as respostas JSON das cotações.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_moedas_tempo_real.py
```
