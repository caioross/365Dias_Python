# 📧 Gerador de Assinatura de E-mail Personalizada

Este script gera um código HTML para uma assinatura de e-mail personalizada, incluindo uma foto e links para suas redes sociais.

## ✨ Funcionalidades
- Criação de uma assinatura de e-mail em formato HTML.
- Inclusão de uma foto personalizada.
- Adição de links para redes sociais (ex: LinkedIn, Twitter, GitHub).
- Saída do código HTML direto no terminal ou em um arquivo.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_assinatura_email.py
```
