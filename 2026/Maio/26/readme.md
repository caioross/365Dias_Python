# 🚀 Monitorador de Frequência do Processador

Este script monitora a variação da velocidade de clock do processador em tempo real, fornecendo uma visão dinâmica da performance do sistema.

## ✨ Funcionalidades
- Monitoramento contínuo da frequência do processador.
- Saída em tempo real da velocidade de clock atual.
- Interface simples e fácil de usar no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python testador_frequencia_processador.py
```
