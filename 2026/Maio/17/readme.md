# 📰 Leitor RSS Terminal

Este script Python exibe as manchetes de um portal de notícias no terminal de forma limpa e rápida, facilitando a visualização das últimas notícias diretamente no seu ambiente de desenvolvimento.

## ✨ Funcionalidades
- Leitura de feeds RSS de um portal de notícias.
- Exibição das manchetes das notícias no terminal.
- Formatação limpa e fácil de ler das informações.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python leitor_rss_terminal.py
```
