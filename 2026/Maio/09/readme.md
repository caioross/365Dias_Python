# 📄 Validador XML Simples

Este script verifica se um arquivo XML está bem formatado, garantindo que todas as tags sejam abertas e fechadas corretamente.

## ✨ Funcionalidades
- Validação de arquivos XML para garantir a correta abertura e fechamento de tags.
- Saída imediata de resultados no terminal indicando se o arquivo está válido ou não.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `seu_arquivo.xml` pelo nome do seu arquivo XML:

```bash
python validador_xml_simples.py seu_arquivo.xml
```
