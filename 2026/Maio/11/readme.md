# 📊 Analisador de Sentimento de Texto

Este script classifica uma frase como positiva, negativa ou neutra baseada em palavras-chave, facilitando a análise de sentimentos em texto simples.

## ✨ Funcionalidades
- Classificação de frases como positivas, negativas ou neutras.
- Utilização de palavras-chave predefinidas para determinar o sentimento.
- Saída direta no terminal com a classificação do sentimento.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python analisador_sentimento_texto.py
```
