# 🚗 Verificador de Placa de Veículo

Este script verifica se uma placa de veículo brasileira segue o padrão antigo (ABC-1234) ou o novo padrão Mercosul (ABC-1234).

## ✨ Funcionalidades
- Verifica se a placa está no padrão antigo (ABC-1234).
- Verifica se a placa está no padrão Mercosul (ABC-1234).
- Retorna uma mensagem indicando se a placa é válida e qual é o padrão.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_placa_veiculo.py
```
