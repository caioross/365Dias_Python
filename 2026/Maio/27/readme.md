# 🍽️ Gerador de Menu Restaurante

Este script Python gera um cardápio formatado em PDF a partir de uma lista de pratos e preços fornecidos em um arquivo CSV. Ele é útil para restaurantes que desejam criar facilmente seus menus digitais.

## ✨ Funcionalidades
- Lê dados de um arquivo CSV contendo pratos e preços.
- Formata os dados em um layout de cardápio profissional.
- Gera um arquivo PDF com o cardápio formatado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas externas necessárias:
  - `fpdf`: Para a geração de PDFs.
  - `pandas`: Para manipulação de dados CSV.

Para instalar as bibliotecas necessárias, execute o seguinte comando:

```bash
pip install fpdf pandas
```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_menu_restaurante.py
```

Certifique-se de que o arquivo CSV com os dados dos pratos esteja na mesma pasta do script ou forneça o caminho correto para o arquivo.
