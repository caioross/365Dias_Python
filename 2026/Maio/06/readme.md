# 🧮 Calculadora Científica Terminal

Este script implementa uma calculadora científica completa com funções avançadas, permitindo realizar cálculos matemáticos complexos diretamente no terminal.

## ✨ Funcionalidades
- Cálculo de seno, cosseno e tangente.
- Cálculo de logaritmo natural e logaritmo base 10.
- Cálculo de raiz quadrada.
- Interface interativa no terminal para seleção de operações.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_cientifica_terminal.py
```
