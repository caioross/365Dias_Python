# 🎨 Gerador Gráfico de Pizza

Este script cria um gráfico de setores (pizza) baseado em categorias e valores fornecidos pelo usuário, facilitando a visualização de dados categóricos.

## ✨ Funcionalidades
- Geração de gráficos de pizza personalizados.
- Solicitação interativa de categorias e valores ao usuário.
- Exibição do gráfico utilizando a biblioteca `matplotlib`.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `matplotlib` instalada. Para instalar, utilize o comando:
  ```bash
  pip install matplotlib
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_grafico_pizza.py
```

3. Siga as instruções no terminal para inserir as categorias e valores.
