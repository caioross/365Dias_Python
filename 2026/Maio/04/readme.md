# 🛡️ Simulador de Combate RPG

Este script Python implementa um sistema de turnos simples para um combate entre dois personagens, onde cada personagem ataca com base em seus atributos de força e sorte.

## ✨ Funcionalidades
- **Turnos Simples:** Dois personagens se enfrentam em turnos alternados.
- **Atributos de Força e Sorte:** Cada personagem tem atributos que determinam a eficácia de seus ataques.
- **Resolução de Combate:** O combate continua até que um dos personagens seja derrotado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_combate_rpg.py
```
