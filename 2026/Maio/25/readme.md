# 📄 Conversor PDF para TXT

Este script Python converte o conteúdo textual de um arquivo PDF para um arquivo de texto simples (.txt), facilitando a extração e manipulação de dados de documentos PDF.

## ✨ Funcionalidades
- Conversão eficiente de texto de PDF para formato .txt.
- Preservação da estrutura básica do texto original.
- Suporte a múltiplos arquivos PDF de uma só vez.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca externa `PyPDF2` para manipulação de PDFs.

Para instalar a biblioteca `PyPDF2`, utilize o seguinte comando:

```bash
pip install PyPDF2
```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `seu_arquivo.pdf` pelo nome do seu arquivo PDF:

```bash
python conversor_pdf_para_txt.py seu_arquivo.pdf
```

Se desejar converter múltiplos arquivos, liste-os separados por espaços:

```bash
python conversor_pdf_para_txt.py arquivo1.pdf arquivo2.pdf arquivo3.pdf
```
