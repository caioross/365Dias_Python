# 📈 Simulador de Venda e Estoque

Este script Python controla a quantidade de produtos em um estoque fictício e alerta quando o nível de estoque está baixo, ajudando a evitar falta de estoque e excesso de estoque.

## ✨ Funcionalidades
- Gerenciamento de estoque fictício.
- Registro de vendas e atualização automática do estoque.
- Alertas quando o estoque de um produto está abaixo de um nível definido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_venda_estoque.py
```
