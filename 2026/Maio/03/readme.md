# 🌧️ buscador_clima_previsao.py

Este script Python fornece uma previsão do tempo para os próximos três dias para uma cidade específica, utilizando uma API de serviços meteorológicos.

## ✨ Funcionalidades
- **Previsão do Tempo:** Exibe a previsão do tempo para os próximos três dias para uma cidade especificada.
- **Interface Simples:** Utiliza uma interface de linha de comando para interação com o usuário.
- **API de Serviços Meteorológicos:** Integra-se a uma API de serviços meteorológicos para obter dados em tempo real.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Bibliotecas Externas:** Requer a biblioteca `requests` para fazer chamadas à API. Pode ser instalada via pip:
  ```bash
  pip install requests
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `nome_da_cidade` pelo nome da cidade desejada:

```bash
python buscador_clima_previsao.py nome_da_cidade
```
