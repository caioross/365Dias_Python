# 🏷️ Gerador de Código de Barras EAN-13

Este script gera uma imagem de código de barras padrão EAN-13 a partir de uma sequência numérica fornecida pelo usuário, facilitando a criação de códigos de barras para produtos.

## ✨ Funcionalidades
- Geração de códigos de barras EAN-13 a partir de uma sequência numérica.
- Salva a imagem gerada no formato PNG.
- Interface simples para inserção da sequência numérica.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca externa `python-barcode` para a geração de códigos de barras.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_codigo_barras.py
```

3. Insira a sequência numérica de 12 dígitos quando solicitado.
4. A imagem do código de barras será salva no diretório atual.
