# 🖱️ Simulador de Cliques Mouse

Este script automático simula cliques do mouse em coordenadas específicas da tela em intervalos de tempo definidos, facilitando tarefas repetitivas ou testes de interface do usuário.

## ✨ Funcionalidades
- Simula cliques do mouse em posições personalizadas.
- Define intervalos de tempo entre cliques para maior precisão.
- Executa em segundo plano sem interrupção do usuário.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
