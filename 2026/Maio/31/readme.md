# 📝 Gerador de Certificados PDF

Este script gera certificados de participação personalizados em massa a partir de uma lista de nomes, facilitando a emissão de documentos para eventos ou programas.

## ✨ Funcionalidades
- Geração automática de certificados em formato PDF.
- Personalização de certificados com nomes de participantes.
- Opção de salvar certificados em uma pasta específica.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas externas necessárias:
  - `reportlab`: Para a criação e manipulação de PDFs.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_certificado_pdf.py
```
