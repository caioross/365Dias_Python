# 📝 Gerenciador de Anotações SQLite

Este script permite que o usuário crie, leia e delete registros de anotações em um banco de dados SQLite, facilitando a gestão de notas pessoais.

## ✨ Funcionalidades
- **Criação de Anotações:** Adiciona novas anotações ao banco de dados.
- **Leitura de Anotações:** Exibe todas as anotações armazenadas no banco de dados.
- **Deleção de Anotações:** Remove anotações específicas do banco de dados.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca padrão do Python (não requer instalações adicionais).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerenciador_anotacoes_sqlite.py
```
