# 🏡 Buscador de CEP Interface

Este script fornece uma interface gráfica para buscar endereços por CEP, com preenchimento automático de informações, facilitando a consulta de dados de localização.

## ✨ Funcionalidades
- Interface gráfica amigável para interação.
- Busca de endereços por CEP.
- Preenchimento automático de informações de endereço.
- Exibição dos resultados diretamente na interface.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas externas necessárias:
  - `tkinter` (biblioteca padrão do Python, não requer instalação adicional)
  - `requests` (para fazer requisições HTTP)

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python buscador_cep_interface.py
```
