# 📈 Simulador de Investimento no Tesouro

Este script compara a rentabilidade bruta entre os investimentos no Tesouro Selic, IPCA e Prefixado em um período específico, ajudando a tomar decisões informadas sobre investimentos.

## ✨ Funcionalidades
- **Comparação de Rentabilidades:** Calcula e compara a rentabilidade bruta dos investimentos no Tesouro Selic, IPCA e Prefixado.
- **Período Personalizado:** Permite definir um período específico para a análise de rentabilidades.
- **Saída Formatada:** Exibe os resultados de forma clara e organizada no terminal.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script, utilizando apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até a pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_investimento_tesouro.py
```
