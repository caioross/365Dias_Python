# 🔍 Validador de Título de Artigo

Este script verifica se um título de artigo segue as normas da ABNT em termos de capitalização, tamanho e ausência de ponto final.

## ✨ Funcionalidades
- Verifica se o título está em maiúsculas corretamente de acordo com as regras da ABNT.
- Garante que o título não exceda um determinado número de caracteres.
- Confirma que o título não termina com um ponto final.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_titulo_artigo.py
```
