# 🖼️ Gerador de Capa de Livro Digital

Este script cria uma imagem JPG simples com o título e o autor centralizados, servindo como uma capa provisória para livros digitais.

## ✨ Funcionalidades
- Geração automática de capas de livros digitais.
- Título e autor centralizados na imagem.
- Saída em formato JPG.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas padrão do Python (built-in), sem necessidade de instalações adicionais.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_capa_livro_digital.py
```
