# 🎨 Gerador de Poesia Aleatória

Este script combina versos e rimas pré-definidas para criar pequenos poemas gerados por computador, adicionando uma toque de criatividade automática à arte poética.

## ✨ Funcionalidades
- Combinação aleatória de versos e rimas para gerar poemas únicos.
- Possibilidade de personalizar o número de versos no poema gerado.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_poesia_aleatoria.py
```
