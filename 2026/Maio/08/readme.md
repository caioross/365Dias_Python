# 🚨 Monitor de Uso de RAM

Este script monitora o uso de RAM do sistema e envia uma notificação ao usuário se o uso ultrapassar um limite definido, ajudando a evitar sobrecargas de memória.

## ✨ Funcionalidades
- Monitoramento contínuo do uso de RAM.
- Definição de um limite de uso de RAM.
- Envio de notificações do sistema quando o limite é ultrapassado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `psutil` para monitoramento do sistema (instalável via pip).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python monitor_consumo_ram.py
```
