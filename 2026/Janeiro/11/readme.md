# 🔄 Reverso de String

Este script recebe uma palavra ou frase e a exibe de trás para frente, ignorando espaços extras. É uma ferramenta simples e útil para manipulação de texto.

## ✨ Funcionalidades
- Inverte a ordem dos caracteres de uma string.
- Ignora espaços extras durante o processo de inversão.
- Exibe o resultado diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
