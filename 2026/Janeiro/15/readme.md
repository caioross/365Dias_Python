# 🔄 Embaralhador de Lista

Este script recebe uma lista de nomes ou itens e os embaralha aleatoriamente utilizando a biblioteca `random`, facilitando a organização ou a distribuição aleatória de itens.

## ✨ Funcionalidades
- Recebe uma lista de nomes ou itens como entrada.
- Embaralha a lista aleatoriamente.
- Exibe a lista embaralhada no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca padrão `random` (não requer instalação adicional).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python embaralhador_de_lista.py
```
