# 📝 Extrator de Iniciais de Nome

Este script extrai e exibe apenas as letras iniciais de um nome completo fornecido pelo usuário, facilitando a geração de siglas ou iniciais de nomes.

## ✨ Funcionalidades
- Solicita ao usuário que insira um nome completo.
- Extrai as letras iniciais de cada palavra do nome.
- Exibe as iniciais em maiúsculas.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
