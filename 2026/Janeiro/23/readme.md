# 🧮 Analisador de Triângulos

Este script verifica se três medidas de lados podem formar um triângulo e, caso afirmativo, determina o tipo de triângulo (equilátero, isósceles ou escaleno).

## ✨ Funcionalidades
- Verifica se três medidas podem formar um triângulo.
- Identifica o tipo de triângulo (equilátero, isósceles ou escaleno).

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python analisador_de_triangulos.py
```
