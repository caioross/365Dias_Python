# 🧮 Ordenador de Lista

Este script recebe uma série de números desordenados e os organiza em ordem crescente e decrescente, facilitando a análise e organização de dados numéricos.

## ✨ Funcionalidades
- Recebe uma lista de números desordenados.
- Organiza os números em ordem crescente.
- Organiza os números em ordem decrescente.
- Exibe os resultados no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python ordenador_de_lista.py
```
