# 🎉 Contador de Vogais

Este script Python recebe uma frase e conta quantas vogais (a, e, i, o, u) existem nela, ignorando maiúsculas e minúsculas.

## ✨ Funcionalidades
- Conta a quantidade de vogais (a, e, i, o, u) em uma frase fornecida pelo usuário.
- Ignora a diferença entre maiúsculas e minúsculas.
- Exibe o resultado no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python contador_vogais.py
```
