# 🔍 Verificador de Palavras Proibidas

Este script verifica se uma frase contém palavras censuradas de uma lista pré-definida no código, ajudando a garantir que o conteúdo esteja de acordo com as políticas de linguagem.

## ✨ Funcionalidades
- Verifica a presença de palavras proibidas em uma frase.
- Retorna uma mensagem indicando se a frase contém palavras proibidas ou não.
- Utiliza uma lista de palavras censuradas definida no código.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
