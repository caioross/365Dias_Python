# 🎲 Simulador de Lançamento de Dados

Este script simula o lançamento de um dado de 6 faces e oferece ao usuário a opção de jogar novamente.

## ✨ Funcionalidades
- Lançamento aleatório de um dado de 6 faces.
- Solicitação ao usuário se deseja realizar outro lançamento.
- Continuação do jogo até que o usuário decida parar.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_lancamento_dados.py
```
