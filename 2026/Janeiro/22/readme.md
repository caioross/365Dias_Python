# 🧮 Calculadora de Gorjetas

Este script calcula o valor da gorjeta com base no total da conta e na porcentagem desejada pelo usuário.

## ✨ Funcionalidades
- Solicita ao usuário o valor total da conta.
- Pede ao usuário a porcentagem da gorjeta que deseja pagar.
- Calcula e exibe o valor da gorjeta e o total a ser pago (conta + gorjeta).

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_de_gorjetas.py
```
