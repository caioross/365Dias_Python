# 🛍️ Gerador de Lista de Compras

Este script permite ao usuário adicionar itens a uma lista de compras e salva o resultado final em um arquivo de texto, facilitando a organização e a gestão de compras.

## ✨ Funcionalidades
- Adicionar itens à lista de compras interativamente.
- Salvar a lista de compras em um arquivo de texto.
- Visualizar a lista de compras atual.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_lista_compras.py
```
