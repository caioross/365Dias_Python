# 🕒 Conversor de Segundos para Horas

Este script converte uma quantidade total de segundos em horas, minutos e segundos formatados, facilitando a conversão de unidades de tempo.

## ✨ Funcionalidades
- Converte uma entrada de segundos em horas, minutos e segundos.
- Saída formatada de forma clara e legível.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_segundos_para_horas.py
```
