# 🧮 Gerador de Sequência de Fibonacci

Este script solicita um número 'n' e exibe os primeiros 'n' elementos da sequência de Fibonacci, uma série matemática onde cada número é a soma dos dois anteriores, começando com 0 e 1.

## ✨ Funcionalidades
- Solicita ao usuário um número 'n'.
- Calcula e exibe os primeiros 'n' números da sequência de Fibonacci.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Nenhuma biblioteca externa é necessária para este script.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_sequencia_fibonacci.py
```
