# 🎉 Boas-Vindas ao Ano Novo

Este script Python exibe uma mensagem de boas-vindas personalizada e calcula quantos dias faltam para o próximo ano, ajudando a preparar-se para o novo ano com antecedência.

## ✨ Funcionalidades
- Exibe uma mensagem de boas-vindas personalizada.
- Calcula e exibe quantos dias faltam para o próximo ano.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python boas_vindas_ano_novo.py
```

## 365 Dias de Python
Este projeto faz parte do desafio 365 Dias de Python, um compromisso de programar e publicar um novo script útil todos os dias durante o ano de 2026.
