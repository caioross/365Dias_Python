# 🧮 Cálculo de Volume de Esfera

Este script calcula o volume de uma esfera a partir do raio informado, resolvendo problemas comuns de geometria.

## ✨ Funcionalidades
- Calcula o volume de uma esfera utilizando a fórmula \( V = \frac{4}{3} \pi r^3 \).
- Solicita ao usuário que insira o raio da esfera.
- Exibe o resultado do cálculo no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_volume_esfera.py
```
