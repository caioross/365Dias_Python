# 🧮 Cálculo de Área de Círculo

Este script calcula a área de um círculo com base no raio fornecido pelo usuário, utilizando a constante Pi da biblioteca `math`.

## ✨ Funcionalidades
- Solicita ao usuário que insira o raio de um círculo.
- Calcula a área do círculo utilizando a fórmula \( \text{Área} = \pi \times \text{raio}^2 \).
- Exibe o resultado do cálculo no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca padrão `math` (não requer instalação adicional).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_area_circulo.py
```
