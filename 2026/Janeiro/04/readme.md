# 🧮 Calculadora de IMC

Este script calcula o Índice de Massa Corporal (IMC) com base no peso e altura fornecidos, classificando o resultado em categorias de saúde.

## ✨ Funcionalidades
- Calcula o IMC utilizando a fórmula padrão: IMC = peso (kg) / altura² (m²).
- Classifica o IMC em categorias como "Abaixo do peso", "Normal", "Sobrepeso" e "Obesidade".
- Solicita ao usuário que insira o peso e altura.
- Exibe o IMC calculado e a categoria correspondente.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_imc.py
```
