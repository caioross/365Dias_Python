# 📈 Calculadora de Desconto de Produto

Este script aplica uma porcentagem de desconto a um valor de produto e exibe o preço final e o valor economizado.

## ✨ Funcionalidades
- Aplica um desconto percentual ao valor de um produto.
- Exibe o preço final após o desconto.
- Calcula e exibe o valor economizado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_desconto_produto.py
```
