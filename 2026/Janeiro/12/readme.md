# 📝 Contador de Caracteres Sem Espaço

Este script Python conta o número total de caracteres em um texto, desconsiderando os espaços em branco na contagem. É útil para análises de texto onde o espaço não deve ser contabilizado.

## ✨ Funcionalidades
- Conta o número total de caracteres em um texto.
- Ignora espaços em branco na contagem.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python contador_caracteres_sem_espaco.py
```
