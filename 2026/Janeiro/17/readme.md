# 🎓 Verificador de Maioridade

Este script solicita o ano de nascimento do usuário e verifica se a pessoa atingiu a maioridade penal e civil, considerando as diferenças entre os países.

## ✨ Funcionalidades
- Solicita o ano de nascimento do usuário.
- Verifica se a pessoa atingiu a maioridade penal (18 anos) e civil (21 anos) no Brasil.
- Informa o resultado diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_maioridade.py
```
