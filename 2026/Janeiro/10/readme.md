# 📅 Verificador de Ano Bissexto

Este script verifica se um determinado ano fornecido pelo usuário é bissexto de acordo com as regras do calendário gregoriano.

## ✨ Funcionalidades
- Verifica se um ano é bissexto.
- Utiliza as regras do calendário gregoriano para a determinação.
- Solicita ao usuário que insira um ano para verificação.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_ano_bissexto.py
```

3. Insira o ano desejado quando solicitado pelo script.
