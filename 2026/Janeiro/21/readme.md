# ⏱️ Cronômetro Simples

Este script inicia um cronômetro no terminal que conta o tempo até que o usuário pressione uma tecla para parar. É uma ferramenta útil para medições de tempo rápidas e precisas diretamente no terminal.

## ✨ Funcionalidades
- Inicia um cronômetro que conta o tempo em segundos.
- Para o cronômetro quando o usuário pressiona qualquer tecla.
- Exibe o tempo decorrido no formato de segundos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python cronometro_simples.py
```
