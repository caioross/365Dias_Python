# 🧮 Conversor de Centímetros para Polegadas

Este script simples e rápido converte valores de centímetros para polegadas, facilitando conversões diárias.

## ✨ Funcionalidades
- Converte facilmente valores de centímetros para polegadas.
- Interface de linha de comando intuitiva.
- Saída direta no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_cm_para_polegadas.py
```
