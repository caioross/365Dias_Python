# 🧮 Gerador de Tabuadas Completa

Este script gera as tabuadas do 1 ao 10 e as salva em um arquivo `.txt`, facilitando a referência e estudo de matemática básica.

## ✨ Funcionalidades
- Geração automática das tabuadas do 1 ao 10.
- Salvamento das tabuadas em um arquivo `.txt`.
- Saída formatada para fácil leitura.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_tabuada_completa.py
```
