# 🧮 Cálculo de Potência de Número

Este script solicita ao usuário a base e o expoente e realiza o cálculo de potência sem utilizar o operador padrão (**), promovendo uma compreensão mais aprofundada do conceito matemático por trás da operação.

## ✨ Funcionalidades
- Solicita a base e o expoente ao usuário.
- Realiza o cálculo de potência sem usar o operador (**).
- Exibe o resultado do cálculo.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Nenhuma biblioteca externa é necessária para este script.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_potencia_numero.py
```
