# 🎲 Gerador de Nicks para Jogos Online

Este script gera nomes de usuários aleatórios para jogos online, combinando adjetivos e substantivos pré-definidos, facilitando a criação de identidades únicas e divertidas.

## ✨ Funcionalidades
- Geração de nomes de usuários únicos e aleatórios.
- Combinção de adjetivos e substantivos para criar nicks criativos.
- Possibilidade de especificar o número de nicks a gerar.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_nicks_games.py
```
