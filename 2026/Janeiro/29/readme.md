# 📧 Validador de Email Básico

Este script verifica se um endereço de email possui um formato válido, contendo o símbolo "@" e um domínio válido após o símbolo.

## ✨ Funcionalidades
- Validação básica de formato de email.
- Verifica a presença do símbolo "@".
- Verifica a existência de um domínio após o símbolo "@".

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_email_basico.py
```
