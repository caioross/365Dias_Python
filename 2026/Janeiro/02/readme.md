# 🧮 Tabuada Interativa

Este script gera a tabuada de um número escolhido pelo usuário, exibindo os resultados de 1 a 10 de forma organizada no terminal.

## ✨ Funcionalidades
- Solicita ao usuário que insira um número entre 1 e 10.
- Calcula e exibe a tabuada do número inserido, de 1 a 10.
- Formata a saída para facilitar a leitura.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Nenhuma biblioteca externa é necessária para este script.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python tabuada_interativa.py
```
