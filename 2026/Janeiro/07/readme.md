# 🧮 Verificador Par ou Ímpar

Este script verifica se um número inteiro digitado pelo usuário é par ou ímpar utilizando o operador de resto da divisão.

## ✨ Funcionalidades
- Solicita ao usuário que insira um número inteiro.
- Verifica se o número é par ou ímpar usando o operador de resto da divisão (`%`).
- Exibe o resultado no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_par_impar.py
```
