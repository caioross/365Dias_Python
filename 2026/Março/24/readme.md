# 🎉 Gerador de Convite de Aniversário

Este script Python cria um arquivo de texto formatado com os dados de uma festa preenchidos pelo usuário, facilitando a organização e a personalização de convites de aniversário.

## ✨ Funcionalidades
- Solicita ao usuário informações sobre o aniversário, como nome do aniversariante, data, local e convidados.
- Gera um arquivo de texto com o convite formatado.
- Permite ao usuário revisar e confirmar os dados antes de salvar o arquivo.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_convite_aniversario.py
```
