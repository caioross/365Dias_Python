# 🔄 Verificador de Palíndromos em Frases

Este script verifica se frases inteiras são palíndromos, ignorando pontuação e espaços, facilitando a análise de textos em busca de propriedades linguísticas interessantes.

## ✨ Funcionalidades
- Verifica se uma frase é um palíndromo, considerando apenas letras e números.
- Ignora espaços e pontuação durante a verificação.
- Saída direta no terminal indicando se a frase é um palíndromo ou não.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_palindromos_frases.py
```
