# 🌍 buscador_clima_cidade.py

Este script consulta o clima atual de uma cidade informada utilizando uma API simples como OpenWeatherMap, facilitando a obtenção de informações meteorológicas rápidas e precisas.

## ✨ Funcionalidades
- Consulta o clima atual de uma cidade específica.
- Utiliza uma API externa para obter dados meteorológicos.
- Exibe informações como temperatura, umidade e condições climáticas.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `requests` instalada. Pode ser instalada via pip:
  ```bash
  pip install requests
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `nome_da_cidade` pelo nome da cidade desejada:

```bash
python buscador_clima_cidade.py nome_da_cidade
```
