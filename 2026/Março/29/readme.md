# 🗂️ Organizador de Pastas Vazias

Este script Python varre um diretório e exclui todas as subpastas que não possuem nenhum arquivo dentro, ajudando a manter a organização e a limpeza do seu sistema de arquivos.

## ✨ Funcionalidades
- Varre um diretório especificado.
- Identifica e exclui subpastas vazias.
- Operações realizadas de forma recursiva, garantindo que todas as subpastas sejam verificadas.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho_do_diretorio` pelo caminho do diretório que deseja organizar:

```bash
python organizador_pastas_vazias.py caminho_do_diretorio
```
