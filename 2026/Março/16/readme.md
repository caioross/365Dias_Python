# 📝 Converte Maiúsculo/Minúsculo

Este script transforma todo o conteúdo de um arquivo de texto para letras maiúsculas, facilitando a padronização de textos.

## ✨ Funcionalidades
- Converte todo o texto de um arquivo para letras maiúsculas.
- Lê e escreve arquivos de texto simples.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python converte_maiusculo_minusculo.py
```
