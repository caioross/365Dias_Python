# 🚗 Calculadora de Consumo de Combustível para Viagem

Este script calcula a quantidade de combustível necessária para uma viagem com base na distância a ser percorrida e no consumo médio do carro.

## ✨ Funcionalidades
- Calcula a quantidade de litros de combustível necessária para uma viagem.
- Solicita ao usuário a distância da viagem e o consumo médio do carro.
- Exibe o resultado diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python consumo_combustivel_viagem.py
```
