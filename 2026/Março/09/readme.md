# 🖼️ Leitor de Metadados de Imagem

Este script extrai informações básicas de uma imagem, incluindo suas dimensões, formato e modo de cor, facilitando a análise rápida de propriedades de imagens.

## ✨ Funcionalidades
- **Extrai Dimensões:** Obtém a largura e altura da imagem.
- **Identifica Formato:** Determina o formato da imagem (por exemplo, JPEG, PNG).
- **Define Modo de Cor:** Informa o modo de cor da imagem (por exemplo, RGB, CMYK).

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca padrão `PIL` (Pillow) para manipulação de imagens. Pode ser instalada via pip:
  ```bash
  pip install pillow
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho/para/sua/imagem.jpg` pelo caminho da imagem que deseja analisar:

```bash
python leitor_metadados_imagem.py caminho/para/sua/imagem.jpg
```
