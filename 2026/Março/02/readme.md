# 🧮 Simulador de Investimento em Poupança

Este script calcula o rendimento mensal de um valor investido em poupança com uma taxa fixa, ajudando a prever o crescimento do investimento ao longo do tempo.

## ✨ Funcionalidades
- Calcula o rendimento mensal de um investimento em poupança.
- Utiliza uma taxa fixa para determinar o crescimento do investimento.
- Exibe o valor acumulado após um determinado número de meses.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_investimento_poupanca.py
```
