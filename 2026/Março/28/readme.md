# 📧 Gerador de E-mails Temporários

Este script gera um endereço de e-mail aleatório para uso em testes rápidos, facilitando a realização de testes de integração sem a necessidade de comprometer informações de contato reais.

## ✨ Funcionalidades
- Geração de endereços de e-mail aleatórios.
- Facilita testes rápidos e seguros.
- Interface simples e direta.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_emails_temporarios.py
```
