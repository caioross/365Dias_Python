# 📝 Conversor Markdown para HTML

Este script converte uma sintaxe básica de Markdown para tags HTML, facilitando a transformação de texto formatado em HTML.

## ✨ Funcionalidades
- Converte cabeçalhos Markdown (`# Título`) para tags HTML (`<h1>Título</h1>`).
- Converte texto em negrito Markdown (`**Negrito**`) para tags HTML (`<strong>Negrito</strong>`).
- Suporta conversão de múltiplos níveis de cabeçalhos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
