# 📞 Extrator de Telefones em Texto

Este script utiliza expressões regulares para identificar e extrair números de telefone de um texto fornecido, facilitando a recuperação de informações de contato.

## ✨ Funcionalidades
- **Extração de Números de Telefone:** Identifica padrões comuns de números de telefone em um texto.
- **Saida Formatada:** Apresenta os números de telefone extraídos de forma clara e organizada.
- **Uso Simples:** Interface de linha de comando fácil de usar para processar textos.

## 🛠️ Pré-requisitos
- **Python 3.x:** Certifique-se de ter a versão mais recente do Python instalada.
- **Bibliotecas Padrão:** O script utiliza apenas bibliotecas nativas do Python, sem necessidade de instalação adicional de pacotes externos.

## 🚀 Como Executar

1. **Navegue até a pasta do script** no seu terminal.
2. **Execute o comando abaixo:**

```bash
python extrator_telefones_texto.py
```
