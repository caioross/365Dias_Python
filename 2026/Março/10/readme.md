# 📂 Comparador de Tamanho de Arquivos

Este script compara dois arquivos e informa qual deles ocupa mais espaço em disco, facilitando a gestão de arquivos e a otimização de armazenamento.

## ✨ Funcionalidades
- Comparação de dois arquivos para determinar o tamanho de cada um.
- Saída clara indicando qual arquivo é maior e por quanto espaço.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
