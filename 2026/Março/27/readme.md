# 🎵 Calculadora de Frequência de Notas Musicais

Este script calcula a frequência em Hertz de uma nota musical baseada em sua oitava, facilitando a geração de sequências musicais ou a análise acústica.

## ✨ Funcionalidades
- Calcula a frequência de uma nota musical específica.
- Suporta todas as notas musicais (C, D, E, F, G, A, B) e suas variações (sustenidos e bemois).
- Permite especificar a oitava da nota.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_frequencia_notas_musicais.py
```
