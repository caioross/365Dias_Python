# 🎨 Gerador de Avatar Simples

Este script gera uma imagem PNG de um quadrado colorido com as iniciais do usuário no centro, facilitando a criação de avatares personalizados.

## ✨ Funcionalidades
- Criação de uma imagem PNG com um quadrado colorido.
- Exibição das iniciais do usuário no centro do quadrado.
- Personalização das cores e tamanho da imagem.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas padrão do Python (não requer instalações adicionais).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_avatar_simples.py
```
