# 🧮 Cálculo de Preço de Venda

Este script calcula o preço de venda de um item com base no custo, impostos e margem de lucro desejada, facilitando a gestão de preços em negócios.

## ✨ Funcionalidades
- Define o preço de venda de um item.
- Considera o custo do item, impostos e margem de lucro.
- Saída do preço de venda calculado no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_preco_venda.py
```
