# 🌐 Verificador de Site Online

Este script verifica se um site específico está online ou fora do ar, fornecendo uma resposta rápida e direta.

## ✨ Funcionalidades
- Verifica a disponibilidade de um site fornecendo uma URL.
- Informa se o site está online ou fora do ar.
- Utiliza uma solicitação HTTP para determinar a disponibilidade.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
