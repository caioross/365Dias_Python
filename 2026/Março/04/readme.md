# 📂 Organizador de Downloads

Este script Python automatiza a organização de arquivos na pasta de Downloads, movendo-os para pastas baseadas no mês e ano de criação. Ideal para manter o ambiente de trabalho limpo e organizado.

## ✨ Funcionalidades
- **Organização Automática:** Move arquivos da pasta de Downloads para pastas específicas criadas com base no mês e ano de criação.
- **Manipulação de Arquivos:** Utiliza funções nativas do Python para lidar com arquivos e diretórios.
- **Facilidade de Uso:** Executável diretamente no terminal com um simples comando.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script, pois utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python organizador_downloads_data.py
```
