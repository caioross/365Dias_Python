# 📧 Extrator de Domínio de E-mails

Este script isola a parte do domínio de uma lista de e-mails, facilitando a análise e organização de dados relacionados a domínios específicos.

## ✨ Funcionalidades
- **Extração de Domínios:** Isola o domínio de uma lista de e-mails fornecida.
- **Saída Formatada:** Exibe os domínios extraídos de forma clara e organizada no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python extrator_dominio_email.py
```
