# 📝 Gerador de Lista de Afazeres CLI

Este script Python é um simples gerenciador de tarefas (To-Do List) que opera totalmente no terminal, permitindo que você crie, visualize e organize suas tarefas de maneira eficiente.

## ✨ Funcionalidades
- Adicionar novas tarefas à lista.
- Visualizar todas as tarefas atuais.
- Marcar tarefas como concluídas.
- Remover tarefas da lista.
- Salvar e carregar a lista de tarefas de um arquivo.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_lista_afazeres_cli.py
```
