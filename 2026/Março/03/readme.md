# 📰 Leitor de Notícias RSS

Este script faz o parsing de um feed RSS simples e exibe os títulos das últimas notícias publicadas, facilitando a leitura rápida e eficiente de feeds de notícias.

## ✨ Funcionalidades
- Parsing de feeds RSS simples.
- Exibição dos títulos das últimas notícias.
- Interface de linha de comando para uso fácil.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python leitor_noticias_rss.py
```
