# 🐶 Calculadora de Idade de Cães

Este script converte a idade de um cachorro para o equivalente em "anos humanos" com base no seu porte, fornecendo uma estimativa mais precisa da idade humana equivalente.

## ✨ Funcionalidades
- **Conversão de Idade:** Calcula a idade humana equivalente de um cachorro com base em seu porte (pequeno, médio, grande).
- **Porte do Cachorro:** Aceita três portes diferentes (pequeno, médio, grande) para uma estimativa mais precisa.
- **Saída Formatada:** Exibe o resultado diretamente no terminal.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_idade_caes.py
```
