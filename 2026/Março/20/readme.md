# 🧮 Calculadora de Horas Trabalhadas

Este script soma as horas e minutos trabalhados no mês para facilitar o controle de ponto, ajudando a gerenciar o tempo de trabalho de forma eficiente.

## ✨ Funcionalidades
- **Soma de Horas:** Calcula a soma total de horas trabalhadas em um mês.
- **Entrada de Dados:** Permite ao usuário inserir horas e minutos trabalhados para cada dia.
- **Saída Formatada:** Exibe o resultado total de horas e minutos trabalhados de forma clara e organizada.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_horas_trabalhadas.py
```
