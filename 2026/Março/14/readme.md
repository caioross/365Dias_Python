# 📊 Filtro de Maiores de 18 Anos CSV

Este script lê um arquivo CSV contendo informações sobre pessoas e gera um novo arquivo CSV apenas com as pessoas que possuem mais de 18 anos.

## ✨ Funcionalidades
- Lê um arquivo CSV de entrada.
- Filtra as pessoas com idade superior a 18 anos.
- Gera um novo arquivo CSV com as pessoas filtradas.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python filtro_maiores_18_csv.py
```
