# 🧹 Limpador de Nomes de Arquivos Especiais

Este script remove caracteres especiais e espaços dos nomes de arquivos para evitar erros em servidores, facilitando a gestão e o acesso aos arquivos.

## ✨ Funcionalidades
- **Remoção de Caracteres Especiais:** Elimina caracteres como `@`, `#`, `$`, `%`, etc., dos nomes de arquivos.
- **Substituição de Espaços:** Substitui espaços por underscores (`_`) para manter a continuidade do nome do arquivo.
- **Iteração em Diretórios:** Processa todos os arquivos em um diretório especificado, renomeando-os conforme necessário.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
