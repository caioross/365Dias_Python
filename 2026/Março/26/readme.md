# 🧮 Validador de Cartão de Crédito

Este script verifica se um número de cartão de crédito é teoricamente válido usando o Algoritmo de Luhn. É uma ferramenta útil para validar números de cartões de crédito rapidamente e com precisão.

## ✨ Funcionalidades
- Verifica a validade de um número de cartão de crédito usando o Algoritmo de Luhn.
- Aceita números de cartão de crédito como entrada.
- Retorna se o número de cartão é válido ou inválido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_cartao_credito.py
```
