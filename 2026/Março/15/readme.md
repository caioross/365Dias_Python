# 🚀 Testador de Velocidade de Internet

Este script realiza um teste básico de download e upload da conexão de internet atual, fornecendo uma visão rápida da velocidade da sua conexão.

## ✨ Funcionalidades
- Medição rápida da velocidade de download e upload da internet.
- Saída dos resultados direto no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `speedtest-cli` instalada. Para instalar, execute o seguinte comando:
  ```bash
  pip install speedtest-cli
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python testador_velocidade_internet.py
```
