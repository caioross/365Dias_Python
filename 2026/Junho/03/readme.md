# 🌟 Gerador de Nuvem de Palavras

Este script gera uma imagem WordCloud (nuvem de palavras) a partir do conteúdo de um livro ou artigo, ajudando a visualizar a frequência das palavras em um texto.

## ✨ Funcionalidades
- Geração de uma imagem WordCloud a partir de um arquivo de texto.
- Personalização de cores e fontes para a nuvem de palavras.
- Remoção de palavras comuns para focar nas palavras mais significativas.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas externas necessárias: `wordcloud`, `matplotlib`, `numpy`.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_nuvem_palavras.py
```
