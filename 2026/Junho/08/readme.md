# 📊 Gerador de Relatório de Sistema

Este script coleta informações de hardware, incluindo CPU, RAM e Disco, e gera um relatório formatado para análise e referência.

## ✨ Funcionalidades
- **Coleta de Informações de Hardware:** Recolhe detalhes sobre a CPU, RAM e Disco do sistema.
- **Relatório Formatado:** Gera um relatório em um formato legível e organizado.
- **Facilidade de Uso:** Interface simples e direta para a execução do script.

## 🛠️ Pré-requisitos
- **Python 3.x:** É necessário ter o Python 3.x instalado na sua máquina.
- **Bibliotecas Padrão:** O script utiliza apenas bibliotecas padrão do Python, por isso não requer instalações adicionais de bibliotecas externas.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_relatorio_sistema.py
```
