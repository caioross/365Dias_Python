# 🛡️ Simulador de Criptografia César

Este script implementa a Cifra de César, uma técnica simples e antiga de criptografia que desloca cada letra de um texto por um número fixo de posições na alfabeto.

## ✨ Funcionalidades
- **Criptografar Mensagens:** Converte mensagens simples em texto cifrado usando a Cifra de César.
- **Descriptografar Mensagens:** Converte mensagens cifradas de volta para texto simples usando a Cifra de César.
- **Configurabilidade:** Permite ao usuário definir o número de posições de deslocamento para criptografia e descriptografia.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_criptografia_cesar.py
```
