# 🌞 Monitor Temperatura CPU

Este script exibe a temperatura atual dos núcleos do processador, ajudando a evitar superaquecimento e garantir o bom funcionamento do sistema.

## ✨ Funcionalidades
- Monitoramento em tempo real da temperatura de cada núcleo do processador.
- Exibição das temperaturas atuais no terminal.
- Alerta se a temperatura ultrapassar um limite definido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `psutil` instalada. Para instalar, utilize o comando:
  ```bash
  pip install psutil
  ```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python monitor_temperatura_cpu.py
```
