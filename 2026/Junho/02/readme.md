# 📧 Extrair Emails TXT

Este script varre um diretório em busca de arquivos de texto e extrai todos os endereços de e-mail únicos, facilitando a recuperação de informações de múltiplos documentos.

## ✨ Funcionalidades
- Varre um diretório especificado para encontrar arquivos `.txt`.
- Extrai endereços de e-mail únicos de cada arquivo.
- Exibe os e-mails encontrados no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python extrair_emails_txt.py
```
