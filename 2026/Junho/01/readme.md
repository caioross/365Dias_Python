# ⏳ cronometro_pomodoro.py

Este script implementa um temporizador Pomodoro, uma técnica de gestão do tempo que consiste em trabalhar por 25 minutos seguidos por uma pausa de 5 minutos. O objetivo é aumentar a produtividade e reduzir a fadiga.

## ✨ Funcionalidades
- **Contagem regressiva:** Timer para 25 minutos de foco e 5 minutos de pausa.
- **Alertas sonoros:** Notificações sonoras no terminal para indicar o início e o fim de cada intervalo.
- **Repetição:** Ciclo automático de 4 intervalos de trabalho e uma pausa longa de 15 minutos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python cronometro_pomodoro.py
```
