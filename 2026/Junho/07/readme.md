# 📝 Validador de PIS/PASEP

Este script verifica a validade do dígito verificador de um número de PIS/PASEP informado, garantindo que o número fornecido seja válido de acordo com as regras estabelecidas pelo governo brasileiro.

## ✨ Funcionalidades
- Validação do dígito verificador de um número de PIS/PASEP.
- Saída clara indicando se o número é válido ou inválido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_pis_pasep.py
```
