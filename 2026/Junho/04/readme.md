# 🧮 Calculadora de Matrizes Soma

Este script Python permite a soma de duas matrizes numéricas de mesma dimensão, fornecidas pelo usuário. Ideal para operações básicas de álgebra linear.

## ✨ Funcionalidades
- **Entrada de Matrizes:** Solicita ao usuário que insira duas matrizes de mesma dimensão.
- **Validação de Dimensões:** Verifica se as matrizes têm as mesmas dimensões antes de realizar a soma.
- **Soma de Matrizes:** Realiza a soma elemento por elemento das duas matrizes.
- **Exibição do Resultado:** Mostra a matriz resultante da soma no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_matrizes_soma.py
```
