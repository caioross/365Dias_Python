# 🧮 Calculadora Geométrica Analítica

Este script calcula a distância entre dois pontos no plano cartesiano e o ponto médio do segmento que os conecta, facilitando cálculos básicos de geometria analítica.

## ✨ Funcionalidades
- Cálculo da distância entre dois pontos no plano cartesiano.
- Determinação do ponto médio de um segmento definido por dois pontos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_geometria_analitica.py
```
