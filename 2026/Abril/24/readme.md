# 🧮 Simulador de Caixa Eletrônico

Este script calcula o menor número de cédulas de 100, 50, 20, 10, 5 e 2 para um valor de saque, facilitando a gestão de pagamentos em caixas eletrônicos.

## ✨ Funcionalidades
- Calcula a menor quantidade de cédulas necessárias para um valor de saque.
- Aceita valores de saque como entrada do usuário.
- Exibe o número de cédulas de cada valor necessário para completar o saque.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_caixa_eletronico.py
```
