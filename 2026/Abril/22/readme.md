# 🔒 Gerador de Senhas Pronunciáveis

Este script Python gera senhas seguras que são compostas por sílabas fáceis de memorizar e pronunciar, aumentando assim a facilidade de uso e a segurança ao mesmo tempo.

## ✨ Funcionalidades
- Geração de senhas seguras e pronunciáveis.
- Personalização da complexidade e comprimento das senhas.
- Saída das senhas geradas diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_senhas_pronunciaveis.py
```
