# 🎲 Simulador de Lançamento de Moeda Viciada

Este script simula o lançamento de uma moeda onde o usuário pode definir uma probabilidade maior para um dos lados (cara ou coroa), permitindo simulações mais realistas ou controladas.

## ✨ Funcionalidades
- Simulação de lançamento de moeda com probabilidade personalizada.
- Definição independente da probabilidade de cara ou coroa.
- Saída do resultado do lançamento no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_lançamento_moeda_viciada.py
```
