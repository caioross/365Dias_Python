# 📅 Validador de Data

Este script valida se uma data fornecida (dia, mês e ano) existe realmente no calendário, considerando os anos bissextos.

## ✨ Funcionalidades
- Validação de datas, incluindo anos bissextos.
- Verifica se o dia, mês e ano formam uma data válida.
- Saída indicando se a data é válida ou inválida.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_data_valida.py
```
