# 📝 Gerador de Configuração JSON

Este script cria um arquivo `config.json` padrão, solicitando as preferências do usuário via terminal. Ideal para configurações rápidas e personalizadas.

## ✨ Funcionalidades
- Solicita ao usuário as preferências necessárias via terminal.
- Gera um arquivo `config.json` com base nas respostas do usuário.
- Facilita a configuração inicial de aplicações ou scripts.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_json_configuracao.py
```
