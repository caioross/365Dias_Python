# 🧮 Verificador de Versão Python

Este script verifica a versão do interpretador Python atualmente em uso e lista todas as bibliotecas instaladas no ambiente, facilitando a gestão e a depuração de ambientes de desenvolvimento.

## ✨ Funcionalidades
- Verifica a versão do interpretador Python.
- Lista todas as bibliotecas instaladas no ambiente Python.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_versao_python.py
```
