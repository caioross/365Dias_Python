# 📈 Contador de Linhas de Código

Este script Python conta quantas linhas de código real existem em um arquivo Python, ignorando comentários e linhas em branco. É uma ferramenta útil para analisar a complexidade e a extensão de um projeto de código.

## ✨ Funcionalidades
- Conta linhas de código reais, excluindo comentários e linhas em branco.
- Pode ser facilmente adaptado para outros tipos de arquivos de código.
- Saída direta no terminal com o número de linhas de código.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python contador_linhas_codigo.py
```
