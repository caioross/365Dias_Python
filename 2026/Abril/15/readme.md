# 📈 Simulador de Investimento CDI

Este script calcula a evolução de um capital baseado em uma taxa percentual do CDI (Certificado de Depósito Interbancário) e um período de tempo em meses.

## ✨ Funcionalidades
- Calcula a evolução de um capital ao longo de um período especificado.
- Utiliza a taxa percentual do CDI para determinar o rendimento.
- Exibe o valor final do investimento após o período definido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_investimento_cdi.py
```
