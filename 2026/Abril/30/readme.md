# 🌐 Tradutor de Texto Online

Este script traduz uma frase do português para o inglês utilizando bibliotecas de tradução automática gratuitas, facilitando a comunicação entre falantes de diferentes línguas.

## ✨ Funcionalidades
- Tradução de frases do português para o inglês.
- Utilização de bibliotecas de tradução automática gratuitas.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `googletrans` (`pip install googletrans==4.0.0-rc1`).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python tradutor_texto_online.py
```
