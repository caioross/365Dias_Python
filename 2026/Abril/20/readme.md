# 🎵 Organizador de Músicas por Artista e Álbum

Este script Python lê as tags ID3 de arquivos MP3 e os organiza em pastas com o nome do artista e álbum, facilitando a gestão e organização de bibliotecas musicais.

## ✨ Funcionalidades
- Leitura das tags ID3 dos arquivos MP3.
- Criação de pastas estruturadas com base no nome do artista e álbum.
- Organização dos arquivos MP3 dentro das pastas correspondentes.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca `mutagen` instalada para manipulação das tags ID3 dos arquivos MP3.

Para instalar a biblioteca `mutagen`, utilize o seguinte comando:

```bash
pip install mutagen
```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python organizador_musicas_artista.py
```
