# 🗂️ Localizador de Arquivos Duplicados

Este script identifica arquivos com o mesmo tamanho e nome em diferentes pastas, auxiliando na limpeza e organização de arquivos duplicados em sistemas de arquivos.

## ✨ Funcionalidades
- **Identificação de Duplicados:** Localiza arquivos com o mesmo nome e tamanho em pastas diferentes.
- **Limpeza de Arquivos:** Ajuda a identificar arquivos que podem ser removidos para liberar espaço ou evitar duplicidade desnecessária.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python localizador_arquivos_duplicados.py
```
