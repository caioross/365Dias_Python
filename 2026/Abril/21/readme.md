# 🧮 Verificador de Ponto Flutuante

Este script compara dois números decimais, considerando uma margem de erro para evitar problemas de precisão comuns em cálculos de ponto flutuante.

## ✨ Funcionalidades
- Compara dois números decimais com uma margem de erro especificada.
- Retorna `True` se os números forem considerados iguais dentro da margem de erro, ou `False` caso contrário.
- Utiliza apenas bibliotecas padrão do Python, sem necessidade de instalações adicionais.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_ponto_flutuante.py
```
