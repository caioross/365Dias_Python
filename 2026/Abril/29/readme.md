# 🔄 Conversor CSV para JSON Customizado

Este script transforma dados de uma planilha CSV em um formato JSON estruturado, permitindo a personalização das chaves do JSON de acordo com as necessidades do usuário.

## ✨ Funcionalidades
- **Transformação de CSV para JSON:** Converte dados de um arquivo CSV em um formato JSON.
- **Customização de Chaves JSON:** Permite ao usuário definir chaves personalizadas para o JSON gerado.
- **Manipulação de Dados:** Facilita a preparação de dados para APIs ou armazenamento em bancos de dados que utilizam JSON.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_csv_para_json_customizado.py
```
