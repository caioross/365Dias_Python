# 🚨 Filtro de Logs de Erros

Este script lê um arquivo de log extenso e extrai apenas as linhas que contêm a palavra "ERROR", facilitando a identificação e análise de erros em logs de aplicativos.

## ✨ Funcionalidades
- Leitura de arquivos de log.
- Filtragem de linhas que contêm a palavra "ERROR".
- Saída das linhas filtradas no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python filtro_logs_erros.py
```
