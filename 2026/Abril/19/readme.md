# 🧮 Calculadora de IMC

Este script Python cria uma interface gráfica simples utilizando Tkinter para calcular o Índice de Massa Corporal (IMC) de um usuário. O IMC é uma medida que relaciona o peso de uma pessoa ao seu quadrado de altura, sendo útil para avaliar o nível de gordura corporal.

## ✨ Funcionalidades
- Interface gráfica amigável para inserção de peso e altura.
- Cálculo do IMC com base nos dados fornecidos.
- Exibição do resultado do IMC e sua classificação (ex: Normal, Sobrepeso, Obesidade).

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Biblioteca Tkinter (incluída na instalação padrão do Python).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_imc_interface.py
```
