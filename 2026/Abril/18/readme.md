# 🛠️ Gerador de Dados Fictícios de Usuários

Este script gera dados fictícios de usuários, incluindo nome, endereço e CPF, para preencher bancos de dados de teste, facilitando a criação de ambientes de desenvolvimento e testes.

## ✨ Funcionalidades
- Geração de dados fictícios de usuários.
- Campos gerados: nome, endereço e CPF.
- Possibilidade de personalizar a quantidade de usuários gerados.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Bibliotecas externas necessárias: `faker` (para geração de dados fictícios).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_fake_user_data.py
```
