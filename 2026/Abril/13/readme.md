# 🗂️ Organizador de Desktop

Este script Python move atalhos e arquivos soltos da área de trabalho para pastas específicas por categoria, ajudando a manter o ambiente de trabalho organizado.

## ✨ Funcionalidades
- **Classificação de Arquivos:** Identifica e classifica arquivos e atalhos na área de trabalho em categorias predefinidas.
- **Movimentação Automática:** Move os arquivos para as pastas correspondentes com base em suas extensões ou nomes.
- **Personalização:** Permite a configuração de categorias e pastas de destino através de um arquivo de configuração.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python organizador_desktop.py
```
