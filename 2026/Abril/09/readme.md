# 📄 Validador de Título de Eleitor

Este script verifica se o formato numérico de um Título de Eleitor brasileiro está de acordo com os padrões estabelecidos pela Receita Federal.

## ✨ Funcionalidades
- Validação do formato numérico do Título de Eleitor.
- Verifica se o número de caracteres é igual a 12.
- Verifica se todos os caracteres são numéricos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_titulo_eleitor.py
```
