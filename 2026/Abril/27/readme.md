# 📋 Gerenciador de Clipboard

Este script monitora a área de transferência e registra o histórico de textos copiados em um arquivo de log, facilitando a recuperação de informações importantes.

## ✨ Funcionalidades
- Monitoriza continuamente a área de transferência.
- Registra automaticamente todos os textos copiados.
- Salva o histórico em um arquivo de log para referência futura.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerenciador_clipboard.py
```
