# 📊 Gerador de Histograma Terminal

Este script cria um gráfico de barras simples usando caracteres ASCII para representar dados numéricos, facilitando a visualização de distribuições de dados diretamente no terminal.

## ✨ Funcionalidades
- Geração de gráficos de barras ASCII a partir de dados numéricos.
- Personalização da altura e largura do histograma.
- Suporte para diferentes conjuntos de dados numéricos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_histograma_terminal.py
```
