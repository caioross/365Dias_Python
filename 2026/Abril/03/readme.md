# 🧩 Verificador de Anagrama de Frase

Este script verifica se duas frases são anagramas uma da outra, desconsiderando espaços. É útil para resolver problemas que envolvem a reorganização de palavras para formar frases idênticas.

## ✨ Funcionalidades
- Verifica se duas frases são anagramas.
- Desconsidera espaços ao comparar as frases.
- Saída direta no terminal indicando se as frases são anagramas ou não.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_anagrama_frase.py
```
