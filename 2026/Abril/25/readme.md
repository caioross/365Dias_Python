# 📊 Monitor de Espaço em Disco

Este script verifica o espaço livre em uma partição do disco e exibe um aviso no terminal caso o espaço livre seja inferior a 10%.

## ✨ Funcionalidades
- Monitora o espaço livre em uma partição específica do disco.
- Exibe um aviso no terminal se o espaço livre for menor que 10%.
- Utiliza apenas bibliotecas padrão do Python.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- Nenhuma biblioteca externa é necessária para este script.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python monitor_espaco_disco.py
```
