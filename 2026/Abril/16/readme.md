# 🎨 Remover Fundo de Imagem

Este script utiliza bibliotecas de IA simples para remover o fundo de uma imagem de forma automatizada, facilitando a edição e compartilhamento de imagens.

## ✨ Funcionalidades
- **Remoção Automática de Fundo:** Utiliza IA para detectar e remover o fundo de imagens de forma precisa.
- **Suporte a Diferentes Formatos:** Aceita e processa vários formatos de imagem comuns.
- **Interface Simples:** Execução direta no terminal sem a necessidade de uma interface gráfica.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Bibliotecas Externas:**
  - `Pillow` para manipulação de imagens.
  - `rembg` para a remoção de fundo utilizando IA.

Para instalar as bibliotecas necessárias, execute o seguinte comando:

```bash
pip install pillow rembg
```

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho/para/sua/imagem.jpg` pelo caminho da imagem que deseja processar:

```bash
python remover_fundo_imagem.py caminho/para/sua/imagem.jpg
```
