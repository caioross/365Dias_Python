# 🧮 Gerador de Sequência de Lucas

Este script calcula os primeiros N termos da Sequência de Lucas, uma sequência matemática semelhante à Sequência de Fibonacci, mas que começa com os números 2 e 1.

## ✨ Funcionalidades
- Geração dos primeiros N termos da Sequência de Lucas.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_sequencia_lucas.py
```
