# 🌐 Verificador de Disponibilidade de Domínio

Este script verifica se um domínio com a extensão `.com.br` ou `.com` está disponível para registro utilizando consultas WHOIS.

## ✨ Funcionalidades
- Verifica a disponibilidade de domínios `.com.br` e `.com`.
- Utiliza consultas WHOIS para determinar se o domínio está disponível.
- Retorna um resultado indicando a disponibilidade do domínio.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_dominio_disponivel.py
```
