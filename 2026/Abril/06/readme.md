# 🧮 Conversor de Polegadas para Centímetros

Este script realiza a conversão de polegadas para centímetros com precisão decimal, facilitando a conversão de medidas entre os sistemas imperial e métrico.

## ✨ Funcionalidades
- Converte valores em polegadas para centímetros.
- Exibe o resultado com precisão decimal.
- Interface simples e direta para uso rápido.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_polegadas_para_cm.py
```
