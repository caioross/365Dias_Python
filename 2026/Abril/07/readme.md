# 📈 Calculadora de Consumo de Energia

Este script estima o custo mensal de um eletrodoméstico com base em sua potência (em Watts) e horas de uso. É uma ferramenta útil para ajudar a monitorar e controlar os gastos com energia elétrica.

## ✨ Funcionalidades
- Calcula o consumo mensal de energia de um eletrodoméstico.
- Solicita ao usuário a potência do eletrodoméstico (em Watts) e as horas de uso diárias.
- Estima o custo mensal com base na tarifa de energia fornecida pelo usuário.
- Exibe o resultado no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculo_consumo_energia.py
```
