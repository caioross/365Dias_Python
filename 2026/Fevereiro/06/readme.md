# 📂 Backup Incremental Simples

Este script Python realiza um backup incremental, copiando apenas os arquivos novos ou modificados de uma pasta de origem para uma pasta de destino de backup. Ideal para economizar espaço e tempo, ele identifica e transfere apenas as alterações necessárias.

## ✨ Funcionalidades
- **Backup Incremental:** Copia apenas arquivos novos ou modificados desde a última execução.
- **Facilidade de Uso:** Interface simples e direta para configuração de origem e destino.
- **Eficiência:** Reduz o tempo de backup ao evitar a cópia de arquivos inalterados.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python backup_incremental_simples.py
```
