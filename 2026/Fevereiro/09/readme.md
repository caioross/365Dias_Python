# 📊 Contador de Frequência de Palavras

Este script analisa um texto fornecido e conta quantas vezes cada palavra aparece, exibindo o ranking das palavras mais comuns. É uma ferramenta útil para análise de texto e processamento de linguagem natural.

## ✨ Funcionalidades
- Análise de texto para contar a frequência de palavras.
- Exibição do ranking das palavras mais comuns.
- Ignora palavras comuns (stop words) para focar em palavras mais relevantes.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python contador_frequencia_palavras.py
```
