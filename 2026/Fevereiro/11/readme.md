# 💳 Simulador de Banco Terminal

Este script Python simula uma conta bancária básica, permitindo realizar operações de depósito, saque e consulta de saldo.

## ✨ Funcionalidades
- **Depósito:** Adiciona um valor ao saldo da conta.
- **Saque:** Subtrai um valor do saldo da conta, se disponível.
- **Consulta de Saldo:** Exibe o saldo atual da conta.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python simulador_banco_terminal.py
```
