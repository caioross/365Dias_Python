# 🔎 Buscador de Palavras em TXT

Este script procura uma palavra específica dentro de um arquivo de texto e indica em quais linhas ela aparece, facilitando a busca de informações em arquivos grandes.

## ✨ Funcionalidades
- Busca uma palavra específica em um arquivo de texto.
- Indica as linhas onde a palavra foi encontrada.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `nome_do_arquivo.txt` pelo nome do seu arquivo e `palavra_a_buscar` pela palavra que deseja procurar:

```bash
python buscador_de_palavras_em_txt.py nome_do_arquivo.txt palavra_a_buscar
```
