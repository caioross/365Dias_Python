# 📋 Lista de Processos Ativos

Este script Python lista os nomes de todos os processos que estão sendo executados no computador no momento, facilitando a monitorização e gestão de processos ativos.

## ✨ Funcionalidades
- **Listagem de Processos:** Exibe o nome de todos os processos ativos no sistema.
- **Simplicidade:** Utiliza apenas bibliotecas padrão do Python, sem necessidade de instalações adicionais.
- **Facilidade de Uso:** Execução rápida e direta no terminal.

## 🛠️ Pré-requisitos
- **Python 3.x:** Este script requer a versão 3.x do Python instalada na sua máquina.
- **Bibliotecas Padrão:** Não requer nenhuma biblioteca externa, utilizando apenas as bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
