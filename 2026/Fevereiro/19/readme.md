# 🔒 Validador de Fortaleza de Senha

Este script analisa a força de uma senha fornecida pelo usuário, avaliando seu tamanho e a variedade de caracteres (letras maiúsculas, minúsculas, números e símbolos), e atribui uma nota de segurança correspondente.

## ✨ Funcionalidades
- Avaliação da força de uma senha com base em seu tamanho e variedade de caracteres.
- Saída de uma nota de segurança que indica o nível de robustez da senha.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python validador_fortaleza_senha.py
```

3. Siga as instruções no terminal para inserir a senha que deseja avaliar.
