# 📂 Organizador de Arquivos por Extensão

Este script Python automatiza a organização de arquivos em uma pasta, movendo-os para subpastas baseadas em suas extensões (por exemplo, `.pdf`, `.jpg`). Ideal para manter seus arquivos em ordem e facilitar a localização.

## ✨ Funcionalidades
- **Identificação de Extensões:** Reconhece automaticamente a extensão de cada arquivo.
- **Criação de Subpastas:** Cria subpastas para cada extensão encontrada.
- **Organização de Arquivos:** Move os arquivos para as subpastas correspondentes às suas extensões.
- **Relatório de Organização:** Gera um relatório no terminal mostrando quantos arquivos foram organizados e para quais pastas.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Ele utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. **Navegue até à pasta do script** no seu terminal.
2. **Execute o comando abaixo:**

```bash
python organizador_arquivos_extensao.py
```
