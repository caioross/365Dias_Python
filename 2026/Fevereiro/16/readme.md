# 🔄 Verificador de Anagramas

Este script verifica se duas palavras fornecidas pelo usuário são anagramas, ou seja, se elas contêm exatamente as mesmas letras em ordens diferentes.

## ✨ Funcionalidades
- Recebe duas palavras como entrada.
- Verifica se as duas palavras são anagramas.
- Exibe o resultado da verificação no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python verificador_anagramas.py
```
