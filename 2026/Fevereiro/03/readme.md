# 🖼️ Renomeador Massa de Fotos

Este script renomeia sequencialmente todos os arquivos de imagem em uma pasta, facilitando a organização e a gestão de múltiplas imagens.

## ✨ Funcionalidades
- Renomeia automaticamente todos os arquivos de imagem em uma pasta.
- Utiliza um padrão de nomeação sequencial (ex: foto_1.jpg, foto_2.jpg).
- Preserva a extensão original dos arquivos.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python renomeador_massa_fotos.py
```
