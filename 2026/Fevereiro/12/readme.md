# 📅 Agendador de Mensagens no Console

Este script permite que o usuário defina um intervalo de tempo e exiba uma mensagem de lembrete no terminal após esse período.

## ✨ Funcionalidades
- Solicita ao usuário que insira uma mensagem e um intervalo de tempo em segundos.
- Aguarda o tempo especificado e exibe a mensagem de lembrete no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python agendador_mensagens_console.py
```
