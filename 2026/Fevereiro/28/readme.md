# 🌟 Gerador de Frases Motivacionais

Este script Python escolhe e exibe uma frase inspiradora aleatória de uma lista interna para começar o dia, ajudando a motivar e inspirar o usuário.

## ✨ Funcionalidades
- Seleção aleatória de frases motivacionais de uma lista interna.
- Exibição da frase escolhida no terminal para iniciar o dia de forma positiva.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_frases_motivacionais.py
```

## 365 Dias de Python
Este projeto faz parte do desafio 365 Dias de Python, um compromisso de programar e publicar um novo script útil todos os dias durante o ano de 2026.
