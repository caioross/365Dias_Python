# 📈 Monitor de Bateria do Notebook

Este script verifica e exibe a porcentagem atual da bateria do seu notebook e informa se o carregador está conectado.

## ✨ Funcionalidades
- Exibe a porcentagem atual da bateria.
- Verifica se o carregador está conectado ao notebook.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python monitor_bateria_notebook.py
```
