# 🔄 Conversor JSON para CSV

Este script converte um conjunto simples de dados no formato JSON para uma tabela organizada em CSV, facilitando a análise e manipulação de dados estruturados.

## ✨ Funcionalidades
- **Conversão de JSON para CSV:** Lê dados em formato JSON e os converte para um arquivo CSV.
- **Manipulação de Dados Simples:** Trata dados JSON de forma eficiente, garantindo que a estrutura seja preservada no formato CSV.
- **Saida Direta no Terminal:** Informa ao usuário sobre o progresso e o sucesso da conversão.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_json_para_csv.py
```
