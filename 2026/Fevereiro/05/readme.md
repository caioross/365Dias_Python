# 🔒 Gerador de Senhas Aleatórias

Este script gera uma senha aleatória de 8 caracteres, alternando entre letras e números, para aumentar a segurança nas suas contas.

## ✨ Funcionalidades
- Criação de senhas seguras e aleatórias.
- Senhas com 8 caracteres, alternando entre letras e números.
- Utiliza apenas bibliotecas padrão do Python.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_senhas_letras_numeros.py
```
