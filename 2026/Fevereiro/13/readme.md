# 🧮 Conversor de Números Romanos para Inteiros

Este script Python converte números romanos (I, V, X, L, C, D, M) para o sistema decimal árabe, facilitando a conversão de números antigos para o formato moderno.

## ✨ Funcionalidades
- Converte números romanos para inteiros.
- Aceita entradas de números romanos válidos.
- Retorna o equivalente numérico no sistema decimal árabe.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_romanos_para_inteiros.py
```
