# 📝 Tradutor Código Morse

Este script converte frases em texto comum para código Morse, utilizando pontos e traços para representar cada letra e número.

## ✨ Funcionalidades
- Converte texto em código Morse.
- Suporta letras do alfabeto inglês e números de 0 a 9.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
