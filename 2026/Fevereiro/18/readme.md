# 📅 Calculadora de Diferença de Datas

Este script calcula a quantidade exata de dias entre duas datas informadas pelo usuário, facilitando o cálculo de intervalos de tempo.

## ✨ Funcionalidades
- Solicita ao usuário duas datas no formato `AAAA-MM-DD`.
- Calcula a diferença exata em dias entre as duas datas.
- Exibe o resultado no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python main.py
```
