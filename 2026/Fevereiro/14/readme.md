# 🗑️ Limpador de Cache Temporário

Este script Python localiza e apaga arquivos da pasta de arquivos temporários do sistema operacional para liberar espaço, ajudando a manter o desempenho do seu sistema.

## ✨ Funcionalidades
- Localiza arquivos temporários no diretório padrão do sistema operacional.
- Apaga os arquivos encontrados para liberar espaço.
- Exibe um relatório dos arquivos deletados e do espaço liberado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python limpador_cache_temporario.py
```
