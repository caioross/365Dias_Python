# 📊 Leitor de Cabeçalho CSV

Este script Python abre um arquivo CSV e exibe apenas os nomes das colunas presentes na primeira linha, facilitando a visualização rápida das informações contidas no arquivo.

## ✨ Funcionalidades
- Leitura de arquivos CSV.
- Exibição dos nomes das colunas da primeira linha do arquivo.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python leitor_cabecalho_csv.py
```
