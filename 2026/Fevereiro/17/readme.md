# 🎨 Gerador de Cartões de Visitas em ASCII

Este script formata informações de contato em uma moldura artística de caracteres ASCII, facilitando a criação de cartões de visitas impressos.

## ✨ Funcionalidades
- Formata informações de contato (nome, cargo, empresa, telefone, email) em um design visualmente atraente.
- Utiliza caracteres ASCII para criar uma moldura única e personalizada.
- Saída direta no terminal para fácil impressão ou compartilhamento.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_cartao_visitas_txt.py
```
