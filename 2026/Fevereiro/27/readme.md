# 📂 Extrator de Extensão de Arquivo

Este script recebe o caminho completo de um arquivo e isola apenas a sua extensão, facilitando a gestão e organização de arquivos em sistemas de computador.

## ✨ Funcionalidades
- **Extrai a extensão de um arquivo:** Identifica e retorna a extensão de um arquivo fornecido.
- **Manipulação de caminhos de arquivos:** Aceita caminhos completos de arquivos, incluindo caminhos relativos e absolutos.
- **Saída direta no terminal:** Exibe a extensão do arquivo no terminal, facilitando a integração em pipelines de automação.

## 🛠️ Pré-requisitos
- **Python 3.x** instalado na sua máquina.
- **Nenhuma biblioteca externa** é necessária para este script. Utiliza apenas bibliotecas padrão do Python.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho_completo_do_arquivo` pelo caminho do arquivo que deseja analisar:

```bash
python extrator_extensao_arquivo.py caminho_completo_do_arquivo
```

**Exemplo:**
```bash
python extrator_extensao_arquivo.py /Users/username/Documents/report.pdf
```
