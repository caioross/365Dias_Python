# 🎨 Gerador de Nomes de Equipes

Este script combina cores e animais aleatoriamente para sugerir nomes criativos para times ou projetos, facilitando a criação de identidades únicas e memoráveis.

## ✨ Funcionalidades
- Geração aleatória de nomes de equipes combinando cores e animais.
- Possibilidade de especificar o número de nomes a gerar.
- Saída formatada diretamente no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python gerador_nomes_equipes.py
```

Para gerar um número específico de nomes, adicione o argumento `-n` seguido pelo número desejado:

```bash
python gerador_nomes_equipes.py -n 5
```
