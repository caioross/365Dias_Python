# 🛡️ Verificador de Integridade de Arquivo

Este script calcula o hash MD5 de um arquivo, permitindo que o usuário verifique se o arquivo foi alterado desde a última verificação.

## ✨ Funcionalidades
- **Cálculo de Hash MD5:** Gera o hash MD5 de um arquivo especificado.
- **Verificação de Integridade:** Permite que o usuário verifique se o arquivo foi alterado comparando o hash atual com um hash anteriormente calculado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho/para/seu/arquivo` pelo caminho do arquivo que deseja verificar:

```bash
python verificador_integridade_arquivo.py caminho/para/seu/arquivo
```
