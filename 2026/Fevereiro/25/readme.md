# 🔍 Verificador de Permissões de Arquivo

Este script verifica se um arquivo específico tem permissões de leitura, escrita ou execução, facilitando a gestão de arquivos e diretórios.

## ✨ Funcionalidades
- Verifica se um arquivo tem permissão de leitura.
- Verifica se um arquivo tem permissão de escrita.
- Verifica se um arquivo tem permissão de execução.
- Saída clara e direta no terminal indicando as permissões do arquivo.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo, substituindo `caminho_do_arquivo` pelo caminho do arquivo que deseja verificar:

```bash
python verificador_permissao_arquivo.py caminho_do_arquivo
```
