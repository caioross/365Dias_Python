# 🧵 Extrator de Links HTML

Este script lê o código de uma página HTML local e extrai todos os links presentes nas tags `<a>`, facilitando a análise e a gestão de links em documentos HTML.

## ✨ Funcionalidades
- Leitura de ficheiros HTML locais.
- Extração de links das tags `<a>` utilizando bibliotecas padrão do Python.
- Saída dos links extraídos no terminal.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python extrator_links_html.py
```
