# 🧮 Conversor de Unidades de Armazenamento

Este script automática e eficiente converte valores entre as principais unidades de armazenamento: Bytes, KB, MB, GB e TB, facilitando a compreensão e a gestão de dados de diferentes tamanhos.

## ✨ Funcionalidades
- **Conversão Automática:** Converte valores entre Bytes, KB, MB, GB e TB.
- **Simplicidade de Uso:** Interface intuitiva que permite a entrada de valores e a seleção das unidades de origem e destino.
- **Saída Formatada:** Resultados exibidos de forma clara e fácil de entender no terminal.

## 🛠️ Pré-requisitos
- **Python 3.x:** É necessário ter o Python 3.x instalado na sua máquina.
- **Bibliotecas Padrão:** O script utiliza apenas bibliotecas padrão do Python, por isso não requer instalações adicionais de bibliotecas externas.

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python conversor_unidades_armazenamento.py
```
