# 🧮 Calculadora de Regra de Três

Este script resolve problemas de regra de três simples, solicitando os três valores conhecidos para calcular o valor desconhecido.

## ✨ Funcionalidades
- Solicita três valores conhecidos para resolver problemas de regra de três.
- Realiza cálculos precisos e exibe o resultado.

## 🛠️ Pré-requisitos
- Python 3.x instalado na sua máquina.
- (Nenhuma biblioteca externa é necessária para este script).

## 🚀 Como Executar

1. Navegue até à pasta do script no seu terminal.
2. Execute o comando abaixo:

```bash
python calculadora_regra_de_tres.py
```
